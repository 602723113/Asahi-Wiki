#### Asahi 前缀解释器拓展 (Asahi Prefix Parser Extension)

```javascript
function wc(){
    print("Hello! My good friend zxy,Long time no see")
}

function main(){
    wc()
    //输出Hello! My good friend zxy,Long time no see
}
```

相信你可以理解函数是干嘛的了，你可以让他有返回值，也可以让他返回一个代码块

还有一种

```javascript
function wc(str){
    print(str)
}

function main(){
    wc("Hello! 碎觉现任")
    //输出"Hello! 碎觉现任"
}
```

这是拥有参数的函数，你需要填入参数

这是比变量还重要的东西！！
